# PROJECT R2C - RESTRICTED ACCESS

An experimental cognitive performance drink website designed with a futuristic, tech-inspired aesthetic.

## Features

- Fully responsive design
- Futuristic cyberpunk aesthetic with scanlines and blinking cursor effects
- Interactive ingredient modal with detailed specifications
- Smooth animations and transitions
- Mobile-friendly layout

## Technologies Used

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (React)
- Babel Standalone for JSX transformation

## How to Run

Simply open `index.html` in any modern web browser.

## Structure

- `index.html` - Main HTML file containing all code
- `README.md` - This file

## Design Elements

- **Typography**: Roboto Mono for body text, Inter for headings
- **Color Scheme**: Dark theme with neon accents
- **Animations**: Fade-in effects, blinking cursor, scanline overlay
- **Components**: Custom SVG icons, bordered tech-style panels
